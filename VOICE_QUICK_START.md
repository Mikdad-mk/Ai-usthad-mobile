## ğŸ”§ Troubleshooting

### Button Not Showing?
- Device doesn't support voice input
- Try restarting the app
- Check device compatibility

### Not Recognizing Speech?
1. Check microphone permission
2. Ensure internet connection
3. Speak more clearly
4. Try again in quieter place

### Partial Text Not Showing?
- Some devices don't support it
- Final text will still work
- This is normal on older devices

### Permission Denied?
1. Go to device Settings
2. Find AI Ustad app
3. Enable Microphone permission
4. Restart app
